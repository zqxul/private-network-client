# HelloWord桌面应用 0.1

## 功能列表
    注册登录
    文本消息
    互联网电话
    直播
    好友

## 技术找
    包管理：NPM
    通信方式：GRPC + NodeJS
    应用搭建：Electron
    界面交互：React
    客户端存储：IndexedDB
    服务端：Golang


## 运行步骤
    npm install
    npm start
